Biography

Hello, I’m Hang Xu, an associate professor in the Intelligent Information Processing Laboratory (IIPL), School of Communication Engineering, Hangzhou Dianzi University (HDU).

My research mainly focuses on object detection and their applications, including spherical image object detection, remote sensing image object detection and 3D object detection.
